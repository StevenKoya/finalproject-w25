About This Project

This final project was created for SI 339: Web Design & Accessibility at the University of Michigan. It is a fan-made educational website that explores the different evolutions of the beloved Pokémon, Eevee.

The site showcases each Eeveelution, its type, combat strengths and weaknesses, and the method required for evolution. It is fully interactive and built using HTML, CSS, JavaScript, and jQuery Mobile. The JavaScript dynamically displays evolution data, allowing users to swipe or click through each evolution card in a user-friendly format.

Special attention was given to visual accessibility and responsive design. The site includes keyboard navigation support, semantic structure, color contrast considerations, and scalable components to ensure it’s easy to use on both desktop and mobile.

Beyond simply listing evolutions, this project is meant to demonstrate the ability to combine functional code with engaging visuals and accessible design practices. Whether you're a Pokémon fan, a beginner web developer, or a curious visitor, we hope this site helps you appreciate the magic and variety of Eevee.

Technologies Used:

  • HTML5 + CSS3
  • jQuery & jQuery Mobile
  • JavaScript DOM Manipulation
  • Responsive & Accessible Design Techniques
